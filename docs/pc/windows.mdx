##  Cài đặt ZaDark với Scoop \{#install-zadark-with-scoop}

:::info
Bạn có thể tìm hiểu thêm về [**Scoop**](https://scoop.sh) trên Website chính thức.
:::

:::warning
Chuyển giao diện Zalo về **Sáng (Light)** trước khi cài đặt ZaDark. <a href="/blog/noti#khắc-phục-lỗi-zadark-trên-zalo-pc-24121-trở-lên" target="_blank">**Xem hướng dẫn**</a>
:::

Bạn mở **Windows PowerShell** và thực hiện theo hướng dẫn bên dưới.

### Cài đặt ZaDark \{#add-zadark-bucket}

```bash
scoop bucket add zadark https://github.com/quaric/scoop-zadark
scoop install zadark/zadark
```

### Sử dụng ZaDark \{#use-zadark-cli}

```bash
# Cài đặt ZaDark
zadark install

# Gỡ cài đặt ZaDark
zadark uninstall

# Kiểm tra phiên bản ZaDark
zadark -v
```

## Các lỗi thường gặp \{#common-errors}

### Không nhìn thấy biểu tượng ZaDark ở thanh bên (Sidebar)

[Zalo chính thức phát hành Dark Mode](https://zadark.com/blog/zalo-chinh-thuc-ra-mat-giao-dien-dark-mode) trên Zalo PC **24.12.1** nên đã xung đột với ZaDark.

Cách nhận biết lỗi:
- Không nhìn thấy biểu tượng ZaDark ở thanh bên (Sidebar); và
- Khi rê chuột vào vị trí bên trên biểu tượng zCloud thì thấy hiện dòng chữ "Cài đặt ZaDark ...", nhưng nhấn vào không có phản hồi.

Cách khắc phục lỗi:

B1. Mở "Cài đặt Zalo" > Chọn "Giao diện (Beta)" > Chọn "Sáng"

### Windows protected your PC

<Image src={require('./img/windows-protected-your-pc-2.png').default} alt="windows-protected-your-pc" isRounded />

ZaDark đến từ nhà phát triển chưa được xác minh (Unknown publisher) nên bạn sẽ gặp thông báo này. <strong style={{ color: 'green' }}>Bạn yên tâm vì ZaDark không chứa virus, mã độc</strong>. Bạn có thể tìm hiểu thêm về [**Chính sách bảo mật**](/privacy-policy).

Cách khắc phục lỗi: Chọn **More info** > **Run anyway** để cài đặt ZaDark.

### EBUSY: resource busy or locked / Vui long tat Zalo PC truoc khi cai dat ZaDark \{#quit-zalo-before-install-zadark}

Lỗi xảy ra khi bạn chưa tắt Zalo PC trước khi cài đặt ZaDark.

Cách khắc phục lỗi:

- B1. Tắt Zalo PC (Xem ảnh GIF bên dưới)
- B2. Chờ biểu tượng Zalo biến mất
- B3. Cài lại ZaDark
